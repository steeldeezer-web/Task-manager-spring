# Task Manager Enterprise (Spring Boot + Web UI)

## 📌 Описание
Полнофункциональное веб-приложение для управления задачами. Проект демонстрирует переход от ручного управления ресурсами к использованию современных фреймворков и декларативных подходов.

## 🚀 Технические особенности:
* **Spring Data JPA & Hibernate**: 
    - Автоматическая генерация схемы БД (`ddl-auto: update`).
    - Использование объектно-реляционного отображения (ORM).
    - Оптимизированные запросы через интерфейсы-репозитории.
* **Domain Model & Relationships**:
    - Реализована связь **Many-to-One** между задачами и пользователями.
    - Использование Enums для управления жизненным циклом задачи.
* **REST & Web Interface**:
    - Реализован **RESTful API** для интеграции с внешними системами.
    - Динамический фронтенд на **Thymeleaf** с использованием **Bootstrap 5** для адаптивного дизайна.
    - Реализован "Soft UI" дизайн через кастомный CSS.
* **Robustness**: 
    - Централизованная обработка ошибок через кастомные исключения (Custom Exceptions).
    - Валидация входных данных на уровне сервисного слоя.

## 🛠 Технологический стек:
* **Framework**: Spring Boot 3.x (Web, Data JPA)
* **Database**: PostgreSQL
* **Templating**: Thymeleaf
* **Frontend**: HTML5, CSS3, Bootstrap 5, Google Fonts (Inter)
* **API Testing**: Postman / Browser Endpoints
